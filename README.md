# Setup

```sh
git clone https://github.com/gcanti/babel-plugin-tcomb-boilerplate.git
cd babel-plugin-tcomb-boilerplate
npm install
npm start # start dev server
```

Then open `localhost:8080`

# Flow

```sh
cd babel-plugin-tcomb-boilerplate
flow # run flow
```
